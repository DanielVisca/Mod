# Privacy Policy for Mod (Chrome Extension)

**Last updated: March 20, 2026**

## 1. Overview

Mod ("we," "the extension") is a Chrome extension that lets you modify websites (e.g., hide elements, apply CSS) with the help of an AI assistant. This policy describes what data the extension uses and where it goes.

## 2. Data We Collect and Use

- **Data you provide:** Your Claude API key (if you use the cloud AI), mod names/descriptions, and the text you type in the extension. This is stored locally in your browser.
- **Data we generate:** Your mods (e.g., CSS, selectors, hide rules) and extension settings (e.g., "Mods on/off") are stored locally so the extension can apply them on the sites you choose.
- **Usage data:** If the extension is configured with analytics (e.g., PostHog), we may collect anonymized usage events (e.g., feature use, errors) to improve the product. No chat content or personal data is included in analytics.

## 3. Where Data Is Stored

- Mods and settings are stored only in your browser (Chrome's local storage). We do not run our own servers that store your mods or API key.

## 4. Third Parties

- **AI provider (Anthropic):** When you use the cloud AI, the text you send (and the context the extension adds, e.g., page structure) is sent to Anthropic's API to generate suggestions. Your API key is used for that request. Anthropic's privacy policy applies to that processing.
- **Analytics (e.g., PostHog):** If analytics are enabled, pseudonymized usage events may be sent to an analytics provider. Events do not include your API key or full chat content.
- We do not sell or rent your data to anyone.

## 5. Permissions

- The extension can run on websites you visit so it can apply your mods and gather minimal page context (e.g., structure, landmarks) for the AI. It does not upload your browsing history. Host and other permissions are used only as described in the Chrome Web Store listing.

## 6. Your Choices

- You can disable or remove the extension at any time. Clearing the extension's storage or uninstalling it removes locally stored data.
- You can choose not to enter an API key and not to use the AI features. Mods and settings remain local.

## 7. Children

- The extension is not directed at children under 13. We do not knowingly collect their personal information.

## 8. Changes

- We may update this policy and will post the new version with an updated "Last updated" date. Continued use of the extension after changes means you accept the updated policy.
